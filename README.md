# Key Code Display

A simple interactive web application built using HTML, CSS, JavaScript, and Bootstrap that displays keyboard key codes dynamically.


## 📌 Project Overview
This project detects the keys pressed by the user and displays their corresponding key codes on the screen in real time.


## 🚀 Features
- Detects keyboard key presses
- Displays key codes dynamically
- Uses JavaScript event handling
- Real-time DOM manipulation
- Responsive UI using Bootstrap



## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 4
